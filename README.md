# ML-Credit-Card-Fraud-2026

This project uses the Credit Card Fraud Detection from Kaggle.

1. **Source:** https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?resource=download
2. **Download:** Download the data manually from the link above or use the script provided.
3. **Placement:** Place the files in the `/data` folder of this repository.
